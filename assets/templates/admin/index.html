Rédaction des articles

Voici quelques astuces à connaître pour rédiger les articles:

  • La miniature: Les dimensions de la miniature sont : width: 900px et height: 300px
  • Images dans l'article: ![Alt text](URL or file path)
    Exemple: ![post_1_1](/static/images/contents/post_1/post_1_1.jpg)
    Il est important de signaler que les images seront sauvegardées dans dossier nommé de cette façon: post_id/

Rédaction des projets

  • L'image du projet : a pour dimensions: Width: 200px et Height: 140px
  • Lien Github du projet: exp: https://github.com/mpiadev/nom_projet.git
